# Portfolio - Iqbal Adil

Ce dossier contient une première base de portfolio en HTML/CSS.

## Fichiers

- `index.html` : structure du site
- `style.css` : design du site
- `images/` : dossier prévu pour ajouter les photos

## Ajouter une photo

1. Mettre l’image dans le dossier `images`
2. Remplacer un bloc `photo-placeholder` par une balise image, par exemple :

```html
<img src="images/bedex.jpg" alt="Participation au salon BEDEX">
```

## Mise en ligne avec GitHub Pages

1. Créer un dépôt GitHub
2. Envoyer les fichiers `index.html`, `style.css` et le dossier `images`
3. Aller dans Settings > Pages
4. Choisir la branche `main`
5. Copier l’URL générée par GitHub Pages
